# MIMO相关的基础知识
## 相关概念
### 1 CSI 相关的概念
CSI(channel state information)信道状态信息，它是一个集合，一个UE上报的集合，包括CQI(channel quality indicator)，PMI(precoding matrix indicator),PTI(Precoding type indicator),RI(rank indicator)

信道状态信息可以分为两部分：信道质量信息（Channel quality information, CQI）和信道
方向息（Channel direction information, CDI）。CQI 多用于用户调度、自适应编码调制及功率 分配，CDI 主要用于设计预编码阵。



### 2 常用的矢量码本（VQC)设计方法
目前的研究方向主要集中在使用基于码本的有限反馈技 术来减小信道状态信息反馈量。在有限反馈系统中，基站侧和用户侧需持有同一码本，依据 最小化用户间干扰准则选择合适的用户，并采用最佳的码字进行波束成形，以提高系统速率。 目前，基于码本的有限反馈多用预编码方案已被广泛研究[18][23]，常用的矢量量化码本 （Vector quantizing code, VQC）的设计方法有三种。
